# SimpleVanish

A lightweight vanish plugin for Spigot/Paper servers.

## Features
- Toggle vanish with /vanish
- Alias with /v
- Fake join and quit messages
- Actionbar vanish status
- View vanished players
- Reload config and messages

## Commands
- /vanish
- /v
- /vanish help
- /vanish list
- /vanish reload

## Permissions
- simplevanish.use
- simplevanish.reload

## Config
The plugin includes:
- config.yml
- messages.yml

## Platform
Tested for Spigot/Paper servers.DS
